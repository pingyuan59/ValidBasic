<H1>Introduction 简介</H1>
ValidBasic is a C++syntax sugar that allows C++ to support most VB syntax.<br>
ValidBasic是C++语法糖，可以让C++支持大部分VB语法。<br>
<br>
<H1>C++Setting Instructions C++设置说明</H1>
<b>1、</b>C/C++>Code Generation Runtime Debug Options/MTD Release Options/MT.<br>
C/C++ > 代码生成 运行库 Debug选/MTD Release选项/MT<br>
<b>2、</b>C/C++>Language conforms to mode selection or not.<br>
C/C++ > 语言 符合模式 选 否<br>
<b>3、</b>C/C++>All Options Additional Options/utf-8.<br>
C/C++ > 所有选项 附加选项 /utf-8<br>
<b>4、</b>The file must be in utf-8 format.<br>
文件必须是utf-8格式<br>
